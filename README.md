# TaskManagerAPI

API for task manager app with user authentication functionality (jsonWebToken). Using Node.js, MongoDB and Jest for tests. Created as a part of a Udemy course.

![Img1](https://github.com/JakubHaj/Task-manager-API/tree/master/ReadmeImg/Postman.PNG | width=910)

## Install

```bash
npm install --save
```

Add valid url for MongoDB and JwtSecretKey

Run application
